# cliptool
A Cross-platform clipboard manager

# Features
- [x] Cross-platform (Electron)
- [ ] Support plain text and image
- [ ] Support Rich Text Format (and HTML)
- [ ] Pinning text
- [ ] Grouping text
- [ ] Open on start up
- [ ] Backup to file
- [ ] Cloud sync
- [ ] Custom hotkeys
